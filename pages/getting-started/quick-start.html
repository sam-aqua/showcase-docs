Quick Start Guide

Comece a usar o Showcase em apenas 5 minutos seguindo este guia passo a passo.

Progresso: 0%

Tempo estimado: 5-10 minutos
Pré-requisitos: Showcase Player instalado (se não tiver, instale aqui)
1

Acesse o Showcase Studio

Faça login na plataforma web

Abra seu navegador e acesse o Showcase Studio através do link fornecido pela Aqua Tecnologia.

Importante: Certifique-se de ter suas credenciais de acesso em mãos.
  1. Digite o endereço do Showcase Studio no navegador
  2. Insira seu usuário e senha
  3. Clique em "Entrar"
2

Crie um Novo Projeto

Configure seu primeiro projeto

Agora vamos criar seu primeiro projeto no Showcase Studio.

  1. No painel principal, clique em "Novo Projeto"
  2. Escolha um nome descritivo para seu projeto
  3. Selecione o template adequado para seu caso de uso:
    • Imobiliário: Para vendas de imóveis
    • Museu: Para exposições culturais
    • Corporativo: Para apresentações empresariais
    • Personalizado: Para casos específicos
  4. Clique em "Criar Projeto"
Dica: Escolha um nome que facilite a identificação do projeto posteriormente.
3

Configure as Categorias

Organize seu conteúdo

As categorias ajudam a organizar seu conteúdo de forma intuitiva para os usuários finais.

  1. No editor do projeto, vá para a seção "Categorias"
  2. Clique em "Adicionar Categoria"
  3. Para cada categoria:
    • Digite um nome claro e descritivo
    • Faça upload de uma imagem representativa
    • Adicione uma descrição breve (opcional)
  4. Organize as categorias na ordem desejada
Formatos suportados: JPG, PNG, GIF (máximo 5MB por imagem)
4

Adicione Conteúdo

Faça upload das suas mídias

Agora é hora de adicionar o conteúdo que será exibido na apresentação.

  1. Selecione uma categoria criada anteriormente
  2. Clique em "Adicionar Mídia"
  3. Faça upload dos arquivos:
    • Imagens: JPG, PNG, GIF
    • Vídeos: MP4, AVI, MOV
    • Documentos: PDF
  4. Para cada arquivo, adicione:
    • Título descritivo
    • Descrição (opcional)
    • Tags para facilitar a busca
Atenção: Arquivos muito grandes podem afetar a performance. Recomendamos otimizar as imagens antes do upload.
5

Publique e Teste

Finalize sua apresentação

Com o conteúdo adicionado, é hora de publicar e testar sua apresentação.

  1. Clique em "Visualizar" para ver como ficará a apresentação
  2. Teste a navegação e interações
  3. Se estiver satisfeito, clique em "Publicar"
  4. Copie o código de sincronização gerado
  5. No Showcase Player, insira o código para sincronizar
Parabéns! Sua primeira apresentação Showcase está pronta!
