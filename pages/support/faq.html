Instalação e Configuração

Como instalar o Showcase Player?

O Showcase Player pode ser instalado em Windows, macOS e Linux:

  1. Acesse a área de downloads no portal do cliente
  2. Baixe a versão correspondente ao seu sistema operacional
  3. Execute o instalador como administrador
  4. Siga as instruções na tela

Ver guia completo de instalação

Quais são os requisitos mínimos do sistema?

Requisitos mínimos:

  • Processador: Intel i3 ou AMD equivalente
  • Memória RAM: 4GB
  • Placa de vídeo: DirectX 11 compatível
  • Armazenamento: 2GB livres
  • Conexão com internet para sincronização

Recomendado: Intel i5, 8GB RAM, placa de vídeo dedicada

Uso Básico

Como criar minha primeira apresentação?

Siga nosso Quick Start Guide para criar sua primeira apresentação em 5 minutos:

  1. Acesse o Showcase Studio
  2. Crie um novo projeto
  3. Configure as categorias
  4. Adicione seu conteúdo
  5. Publique e sincronize

Seguir o Quick Start Guide

Como sincronizar conteúdo entre Studio e Player?

A sincronização é feita através de códigos únicos:

  1. No Studio, publique seu projeto
  2. Copie o código de sincronização gerado
  3. No Player, vá em Configurações > Sincronização
  4. Cole o código e clique em "Sincronizar"

A sincronização acontece automaticamente a cada 30 minutos ou pode ser forçada manualmente.

Questões Técnicas

Quais formatos de arquivo são suportados?

Imagens: JPG, PNG, GIF, BMP, TIFF

Vídeos: MP4, AVI, MOV, WMV, FLV

Documentos: PDF

Áudio: MP3, WAV, AAC

Tamanho máximo: 100MB por arquivo

Como configurar telas múltiplas?

O Showcase suporta configurações multi-tela:

  1. Conecte todas as telas ao computador
  2. Configure-as como "Estender" no Windows
  3. No Player, vá em Configurações > Display
  4. Selecione a tela principal para o Showcase
  5. Configure a resolução adequada

Cobrança e Licenças

Como funciona o modelo de licenciamento?

O Showcase utiliza licenciamento por dispositivo:

  • Licença Básica: 1 dispositivo, recursos essenciais
  • Licença Profissional: 3 dispositivos, recursos avançados
  • Licença Enterprise: Dispositivos ilimitados, suporte premium

Entre em contato para planos personalizados.

Solução de Problemas

O Player não está sincronizando. O que fazer?

Verifique os seguintes pontos:

  1. Conexão com internet ativa
  2. Código de sincronização correto
  3. Firewall não está bloqueando o Player
  4. Horário do sistema está correto

Se o problema persistir, entre em contato com nosso suporte.

A apresentação está lenta. Como otimizar?

Para melhorar a performance:

  • Otimize imagens (máximo 1920x1080)
  • Comprima vídeos para web
  • Reduza o número de elementos por tela
  • Feche outros programas desnecessários
  • Verifique se os drivers da placa de vídeo estão atualizados
